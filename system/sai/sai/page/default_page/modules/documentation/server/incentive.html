Point system

Measurement data will be rewarded by points. These points are used for temporal (cf. global ranking) and location-based (cf. Titles) ranking.

Titles Titles are location-based incentive ranking system, i.e. users get points for specific locality (e.g. countries, cities) when they measured inside the corresponding locality. The best "Noisemappers" of a locality obtain a special title (e.g. Mayor, President).

Bonus Areas Bonus areas are pre-defined circular areas. If a user measures inside these areas, his points for the measurement data will be multiplied with a specific factor defining by the bonus area.
